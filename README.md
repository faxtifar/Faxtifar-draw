# Faxtifar-draw

Полнофункциональное веб-приложение для рисования на основе Excalidraw. Позволяет создавать эскизы, диаграммы и рисунки с различными инструментами.

## Возможности

- **Инструменты рисования**:
  - ✏️ Карандаш - свободное рисование
  - 📏 Линия - рисование прямых линий
  - ▭ Прямоугольник - рисование прямоугольников
  - ⭕ Круг - рисование кругов
  - → Стрелка - рисование стрелок
  - A Текст - добавление текста
  - 🧹 Ластик - стирание

- **Функции**:
  - Выбор цвета для линий и фона
  - Регулировка размера кисти
  - Отмена (Ctrl+Z) и повтор (Ctrl+Y)
  - Очистка холста
  - Скачивание рисунка как PNG
  - Сохранение рисунка в браузере (localStorage)
  - Клавиатурные сокращения

## Клавиатурные сокращения

| Клавиша | Действие |
|---------|----------|
| V | Выделение |
| P | Карандаш |
| L | Линия |
| R | Прямоугольник |
| C | Круг |
| A | Стрелка |
| T | Текст |
| E | Ластик |
| Ctrl+Z | Отменить |
| Ctrl+Y | Повторить |

## Использование

1. Откройте `index.html` в веб-браузере
2. Выберите инструмент из панели инструментов
3. Рисуйте на холсте
4. Используйте кнопки для сохранения или скачивания рисунка

## Установка

Просто откройте файл `index.html` в любом современном веб-браузере. Никаких зависимостей не требуется!
