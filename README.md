# Faxtifar-draw

Профессиональное веб-приложение для рисования на основе HTML5 Canvas. Позволяет создавать эскизы, диаграммы и рисунки с различными инструментами и полной поддержкой истории.

## 🎨 Возможности

### Инструменты рисования
- **✏️ Карандаш (P)** - свободное рисование с гладкими линиями
- **📏 Линия (L)** - рисование прямых линий
- **▭ Прямоугольник (R)** - рисование прямоугольников
- **⭕ Круг (C)** - рисование кругов
- **➜ Стрелка (A)** - рисование стрелок с наконечниками
- **A Текст (T)** - добавление текста на холст
- **🧹 Ластик (E)** - стирание с регулируемым размером
- **🪣 Заливка (F)** - заливка областей одного цвета
- **⬚ Выделение (V)** - инструмент выделения

### Функции управления
- **🎨 Выбор цвета** - полная палитра RGB
- **🎨 Цвет фона** - изменение цвета холста
- **📏 Размер кисти** - регулировка от 1 до 100 пикселей
- **👁️ Прозрачность** - контроль прозрачности от 0 до 100%
- **↶ Отмена (Ctrl+Z)** - отмена последнего действия
- **↷ Повторить (Ctrl+Y)** - повтор отменённого действия
- **🗑️ Очистить** - полная очистка холста
- **⬇️ Скачать** - экспорт рисунка как PNG
- **💾 Сохранить** - сохранение в браузер (localStorage)

## 🚀 Использование

1. Откройте `index.html` в веб-браузере
2. Выберите инструмент из левой панели или используйте горячие клавиши
3. Рисуйте на белом холсте
4. Используйте кнопки для сохранения или скачивания рисунка

## 💻 Технологии

- **HTML5** - структура приложения
- **CSS3** - современный дизайн с градиентами и анимациями
- **Vanilla JavaScript** - полная функциональность без зависимостей
